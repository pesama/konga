translateComplex
filter in module konga

Description

translateComplex

It receives a String (input) and extra (if any), and it returns a complex field to be translated using standard translate filter.

Usage

In HTML Template Binding

{{ translateComplex_expression | translateComplex[:extra] }}

In JavaScript

$filter('translateComplex')(input[, extra])

Parameters

ParamTypeDetails
inputstring

Defines the name of label of the field

extra
(optional)
Object

Defines the extra expression object for filter
