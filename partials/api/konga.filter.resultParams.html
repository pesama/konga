resultParams
filter in module konga

Description

resultParams

It receives an array of fields and metadata , and it returns another array of fields containing the value of sorting field equal 'asc'.

Usage

In HTML Template Binding

{{ resultParams_expression | resultParams:metadata }}

In JavaScript

$filter('resultParams')(fields, metadata)

Parameters

ParamTypeDetails
fieldsArray

Defines an array of fields of the entity

metadataObject

Defines the metadata of the entity
