scroll
filter in module konga

Description

scroll

It receives an array of objects (i.e. input), and it returns another array containing elements of the old one arranging the index from 0 to limit.

Usage

In HTML Template Binding

{{ scroll_expression | scroll:limit }}

In JavaScript

$filter('scroll')(input, limit)

Parameters

ParamTypeDetails
inputObject

Defines an array of objects

limitNumber

Defines the number of object getting from input.
