searchParams
filter in module konga

Description

searchParams

It receives an entity metadata information, and returns all its fields that could be used for searching purposes.

Usage

In HTML Template Binding

{{ searchParams_expression | searchParams[:metadata] }}

In JavaScript

$filter('searchParams')(fields[, metadata])

Parameters

ParamTypeDetails
fieldsArray

Defines an array of fields of the entity

metadata
(optional)
Object

Defines the metadata of the entity
