extended
filter in module konga

Description

extended

It receives an array of fields in metadata and a name of field which are checked, and it returns true/false (i.e. isExtended or not).

Usage

In HTML Template Binding

{{ extended_expression | extended:name }}

In JavaScript

$filter('extended')(metadata, name)

Parameters

ParamTypeDetails
metadataArray

Defines the array of fields of metadata

namestring

Defines the name of checked field
