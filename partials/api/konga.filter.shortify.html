shortify
filter in module konga

Description

shortify

It receive a String (i.e. input) and length, and it returns another one containing extract characters from the old one arranging the index from 0 to length.

Usage

In HTML Template Binding

{{ shortify_expression | shortify:length }}

In JavaScript

$filter('shortify')(input, length)

Parameters

ParamTypeDetails
inputstring

Defines the name of field

lengthNumber

Defines the number of extract characters
