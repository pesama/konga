api
service in module konga

Description

This factory connects the source to a factory, depending on the type of entity that's being looked for.

Methods

  • get(id, success, error)

    Returns an entity identified with the unique id provided

    Parameters
    ParamTypeDetails
    id*

    The unique id for the entity

    successFunction

    Function to call if the operation successes.

    (default: undefined)

    errorFunction

    Function to call if the operation fails.

    (default: undefined)

    Returns
    Object

    The $resource with that entity

  • search(query, offset, limit, success, error)

    Search for entities filtering with the input query

    Parameters
    ParamTypeDetails
    queryObject

    Query to filter with (sent via GET parameters, so they must be serializable, and serialized).

    offsetNumber

    Defines where to place the starting cursor.

    limitNumber

    Defines how many results to get.

    successFunction

    Function to call if the operation successes.

    (default: undefined)

    errorFunction

    Function to call if the operation fails.

    (default: undefined)

    Returns
    Array

    Array containing all results. Each result is a $resource that contains an entity which matched the search criteria.
