FieldSet
service in module Metadata

Description

Field sets define groups of fields, who are normally intended to be bound together into a view. Field sets are defined at entity-level, and thereafter its fields could reference the field set they belong to by declaring its name.

Usage

FieldSet(name, configuration);

Parameters

ParamTypeDetails
namestring

Defines the name of the field set. This name must be unique within the entity-defined field sets. Fields that belong to a field set must declare this name (exactly) and it will be bound to the field set on the forms.

configurationObject

ConfigurationParam Configuration for the field set. There is only one standard param, whose key must be 'view', and the value a valid mapped view for a custom form. This parameter is used when the form type is set to CUSTOM_TABBED.
