selectData
filter in module konga

Description

#selectData It receives metadata, an array of entities and configuration (i.e. boolean), and it returns an array of objects containing data from these parameters (i.e. entities) for selecting purpose.

Usage

In HTML Template Binding

{{ selectData_expression | selectData:entities:configuration }}

In JavaScript

$filter('selectData')(metadata, entities, configuration)

Parameters

ParamTypeDetails
metadataObject

Defines the metadata of the entity

entitiesObject

Defines the an array of entities to manage

configurationObject

Defines an object containing added and selected field.
