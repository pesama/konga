FormStyles
service in module Metadata

Description

Form styles configure the rendering relationships between each input and its label, as well as how to render fields among the others.

Properties

  • HORIZONTAL

    In horizontal mode, label and field are rendered inline. However, fields are on separate lines.

  • INLINE

    Inline mode. All fields and their labels are rendered inline, following the HTML cascade flow.

  • STANDARD

    Block format. Using this style, label and input will be rendered in separate lines.
