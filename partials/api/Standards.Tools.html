Tools
service in module Standards

Description

Konga comes with a set of built-in tools to access quick metadata-parsing features, along with other handy things needed to manage metadata and other common features. This tools are allocated in a service called util. However, you can find here all the tools included, how to use them...

  • getMetadata(name)

    This method will give you the metadata definition of the entity named with the param name provided.

    angular.module('myAwesomeApp')
  .something(['util', function(util) {
    // Here you got your metadata
    var metadata = util.getMetadata('demo-entity');
  }]);
    Parameters
    ParamTypeDetails
