selectedFields
filter in module konga

Description

selectedFields

It receives parameters such as real, name, apiNames(if any), source, and it returns an array of fields in nested fields for rendering purpose.

Usage

In HTML Template Binding

{{ selectedFields_expression | selectedFields:names[:apiNames][:source] }}

In JavaScript

$filter('selectedFields')(real, names[, apiNames][, source])

Parameters

ParamTypeDetails
realArray

Defines the array fields of entity to manage

namesArray

Defines the array of field name

apiNames
(optional)
Array

Defines the array of apiName of entities

source
(optional)
Array

Defines the field to manage
