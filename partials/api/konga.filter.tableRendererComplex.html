tableRendererComplex
filter in module konga

Description

tableRendererComplex

It receives value and metadata , and it returns the value of column which shows the result of complex fields base on these parameters.

Usage

In HTML Template Binding

{{ tableRendererComplex_expression | tableRendererComplex:value }}

In JavaScript

$filter('tableRendererComplex')(metadata, value)

Parameters

ParamTypeDetails
metadataObject

Defines the metadata of the entity to manage

valueObject

Defines the entity of each row
