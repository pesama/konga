mapField
filter in module konga

Description

This filter helps you getting the value of a field name specified.

Basic filtering

entity: {
  id: 2,
  name: 'test'    
},
field: 'name'

<span>{{ entity | mapField:field }}</span>

Result: 'test'

This filter allows you to get the value of a field within an instance of an entity. It's used all across native forms for rendering the values of the fields.

Complex rendering

entity: {
  id: 2,
  name: 'test',
  owner: {
    firstName: 'John',
    lastName: 'Doe'
  }
},
field: 'owner.firstName'

<span>{{ entity | mapField:field }}</span>

Result: 'John'

Field unmapping

This filter is made for reading values out from fields. You have a tool designed for the opposite purpose - i.e. fieldMapper.

Usage

In HTML Template Binding

{{ mapField_expression | mapField:field }}

In JavaScript

$filter('mapField')(entity, field)

Parameters

ParamTypeDetails
entityObject

Defines the entity to manage

fieldObject

Defines the field to manage
