DataTypes
service in module Metadata

Description

Data types define the typology of the field and its value. Depending on the data type chosen for a field, there would be different operations applied, validation rules, scaffolding and field mapping processes, etcetera.

Properties

  • BOOLEAN

    Defines a boolean (yes|no) field. There are a couple of field types related to this data type. With BOOLEAN field type you will have a checkbox rendered input on search forms, that will allow you to search (true|false|both) values. There's also a cool switch input to simulate an on/off button instead of boring radio buttons.

  • COMPLEX

    Complex data types reference an inner entity. This is useful to render and interact with relationships, as Konga will have full comprehension of the values for the field. For using this data type you will need to define also the complexType this element refers to. And you have also handy tools for filtering, querying, selecting data sources... See the data type documentation for all the possible features.

  • DATE

    Date types. This field allows values from timestamps to formal-date-strings (i.e. 'yyyy-mm-dd'). This data type could be rendered as a date or a date&time field, depending on whether you need time accuracy or just day-based dates.

  • FILE

    This data type represents a physical file that travels through Konga forms. Fields annotated with this type are usually Strings that links to the file location, although you can use base64-encoded files, or any other flavor you'd like.

  • NUMBER

    Defines a numeric field. Numbers allow Integers, Doubles, Longs... Any numeric input. If you use this data type along with NUMBER field type, it will easily allow you to manage numbers, and their integrity, configure range search configuration, etcetera.

  • STRING

    Defines a plain String input. Using this data type along with some validation rules, and using REGEXP validators, you can easily configure the String field as a valid email address - or any other formal-syntax String.
