updateParams
filter in module konga

Description

updateParams

It receives an array of fields , metadata and category, and it returns all entity's fields that could be used for updating purposes.

Usage

In HTML Template Binding

{{ updateParams_expression | updateParams:metadata:the }}

In JavaScript

$filter('updateParams')(fields, metadata, the)

Parameters

ParamTypeDetails
fieldsArray

Defines an array of fields of the entity.

metadataObject

Defines the metadata of the entity

theObject

category of the entity
